

## 1. `df` — 查看整体磁盘使用情况

| 命令            | 说明                      |
| ------------- | ----------------------- |
| `df`          | 显示所有文件系统的磁盘使用情况（单位为 KB） |
| `df -h`       | 以人类可读格式显示（GB/MB）        |
| `df -T`       | 显示文件系统类型                |
| `df -i`       | 显示 inode 使用情况           |
| `df -h /home` | 查看指定目录所在分区的使用情况         |

**示例输出：**

```
Filesystem      Size  Used Avail Use% Mounted on
/dev/sda1        50G   20G   28G  42% /
tmpfs           1.9G     0  1.9G   0% /dev/shm
```

## 2. `du` — 查看目录或文件占用空间

|命令|说明|
|---|---|
|`du`|显示当前目录及子目录占用的磁盘空间（单位为 KB）|
|`du -h`|以人类可读格式显示|
|`du -s`|显示总占用空间汇总|
|`du -sh *`|查看当前目录下各文件/文件夹占用空间|
|`du -h /var/log`|查看指定目录占用空间|
|`du -ah`|显示所有文件和目录的详细空间使用|

**示例输出：**

```
4.0K    ./bin
2.3M    ./log
1.8G    ./data
```
