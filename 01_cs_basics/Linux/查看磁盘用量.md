

## 1. `df` — 查看整体磁盘使用情况

| 命令            | 说明                      |
| ------------- | ----------------------- |
| `df`          | 显示所有文件系统的磁盘使用情况（单位为 KB） |
| `df -h`       | 以人类可读格式显示（GB/MB）        |
| `df -T`       | 显示文件系统类型                |
| `df -i`       | 显示 inode 使用情况           |
| `df -h /home` | 查看指定目录所在分区的使用情况         |

**示例输出：**

```
Filesystem      Size  Used Avail Use% Mounted on
/dev/sda1        50G   20G   28G  42% /
tmpfs           1.9G     0  1.9G   0% /dev/shm
```

---

## 2. `du` — 查看目录或文件占用空间

|命令|说明|
|---|---|
|`du`|显示当前目录及子目录占用的磁盘空间（单位为 KB）|
|`du -h`|以人类可读格式显示|
|`du -s`|显示总占用空间汇总|
|`du -sh *`|查看当前目录下各文件/文件夹占用空间|
|`du -h /var/log`|查看指定目录占用空间|
|`du -ah`|显示所有文件和目录的详细空间使用|

**示例输出：**

```
4.0K    ./bin
2.3M    ./log
1.8G    ./data
```

---

## 3. 常见组合命令

|命令|用途|
|---|---|
|`du -sh *|sort -h`|
|`df -hT`|查看磁盘空间及文件系统类型|
|`du -sh /path/to/dir`|快速查看某目录总占用空间|



在 Linux 中，你可以使用以下命令查看某个目录下的 **文件/文件夹大小**（并以人类可读的方式显示，例如 KB、MB、GB）。

---

# ✅ **1. 查看当前目录下所有文件/文件夹大小**

```bash
du -h --max-depth=1
```

说明：

- `du`：disk usage
    
- `-h`：human readable（KB/MB/GB）
    
- `--max-depth=1`：只显示当前目录的第一层，不深入子目录
    

---

# ✅ **2. 查看指定目录大小**

```bash
du -sh /path/to/dir
```

示例：

```bash
du -sh /home/user/projects
```

说明：

- `-s`：summary，只显示总大小
    
- `-h`：人类可读
    

---

# ✅ **3. 查看某目录下所有子目录大小（人类可读）**

```bash
du -h /path/to/dir --max-depth=1
```

---

# ✅ **4. 仅查看某个文件的大小**

```bash
du -h filename
```

---

# 🔥 **补充：用 ls 查看文件大小（但不适用文件夹）**

```bash
ls -lh
```

---

# 🔍 **最常用的组合命令（你 99% 需要的就是这两个）**

### **查看当前目录下每个子目录的大小**

```bash
du -h --max-depth=1
```

### **查看整个目录的总大小**

```bash
du -sh .
```
