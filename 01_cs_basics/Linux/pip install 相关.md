### `pip install -e`（editable mode）说明

**editable mode**（可编辑模式）是一种开发者常用的安装方式。它让你在本地修改源代码后，无需重新安装包即可立即生效。

#### 工作机制

`pip install -e` 会在当前 Python 环境中创建一个指向源代码目录的符号链接（或 `.pth` 文件），而不是复制整个包内容。这样 Python 运行时始终加载最新的源代码。

#### 示例

假设你的项目结构如下：

```
myproject/
├── setup.py
└── mypackage/
    └── __init__.py
```

在项目根目录下执行：

```bash
pip install -e .
```

此时，`mypackage` 会被安装为可编辑包。你在 `mypackage/` 中修改任何文件后，再运行：

```bash
python -m mypackage
```

程序会自动使用更新后的代码，无需再次安装。

#### 典型应用场景

- Python 包或库的开发阶段
    
- 参与开源项目时本地调试代码
    
- 与测试框架（如 `pytest`）结合使用，方便持续集成和自动化测试
