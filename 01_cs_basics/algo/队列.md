
1. 滑动窗口 + 队列

队列维护滑动窗口中的最大/最小值，左侧指针收缩的时候可以及时 update 当前队列中的最大最小值（实际上就是左侧开口的单调栈）

[239](https://leetcode.cn/problems/sliding-window-maximum/description/)

```python
class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        q = deque()
        ans = []

        for i in range(len(nums)):
            while q and nums[q[-1]] < nums[i]:
                q.pop()
            q.append(i)

            if q[0] < i - k + 1:
                q.popleft()
            
            if i - k + 1 >= 0:  # initials
                ans.append(nums[q[0]])
        return ans
```
