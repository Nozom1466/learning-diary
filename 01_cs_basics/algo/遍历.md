
两个变量就是遍历右边维护左边，左边维护可以使用哈希表：

[1](https://leetcode.cn/problems/two-sum/description/)
```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        freq = {}
        for i in range(len(nums)):
            if target - nums[i] in freq.keys():
                return [freq[target - nums[i]], i]
            freq[nums[i]] = i
            
```




三个变量的遍历，优先遍历中间的，这样自然的分成左右两个部分：

[2909](https://leetcode.cn/problems/minimum-sum-of-mountain-triplets-ii/)
```python
class Solution:
    def minimumSum(self, nums: List[int]) -> int:
        min_rev = []
        miner = float('inf')
        for i in range(len(nums)):
            if nums[len(nums) - 1 - i] < miner:
                miner = nums[len(nums) - 1 - i]
            min_rev.append(miner)
        min_rev.reverse()
        
        miner = float('inf')
        ans = float('inf')
        for j in range(1, len(nums) - 1):
            miner = min(miner, nums[j - 1])
            if nums[j] > miner and nums[j] > min_rev[j + 1]:
                ans = min(ans, nums[j] + miner + min_rev[j + 1])
        return ans if ans != float('inf') else -1
        
```
